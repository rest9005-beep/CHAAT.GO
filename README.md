# CHAT.GO (Node 24) — DM + Search + Attachments

Добавлено:
1) ЛС (личные сообщения) — клик по аве/нику открывает диалог  
2) Поиск по username (в левом блоке чата)  
3) В чате показывается аватар  
4) Вложения: фото/файлы + голосовые (запись микрофона в браузере)  
5) Описание профиля + базовая цензура (замена на ****)

## Запуск
```bat
npm install
copy .env.example .env
npm start
```
Открыть: http://localhost:3000

## Лимиты
- Upload: до 25MB
- Описание: 280
- Сообщение: 1200


Новая навигация: вкладки **Общий чат / Личные чаты / Поиск**.
